# Overview

Shiori is a lightweight, extensible, multi-format eBook reader and conversion system designed to provide a distraction-free experience.

It combines reading tools, metadata editing, format conversion, RSS integration, and theme customization into a cohesive interface.

The philosophy is minimal surface, maximum clarity.

# Core Capabilities

- Multi-format support (PDF, TXT, extensible architecture)
- Format conversion engine
- Two-page reading mode
- Table of contents navigation
- Keyword search inside books
- Library-wide search
- Metadata editing
- Page-level navigation
- Light / Dark themes
- RSS feed reader

# Installation

```bash

cd Shiori
```
