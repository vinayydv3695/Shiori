## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-ping`
- `allow-select-folder`
- `allow-select-files`
- `allow-solve-cloudflare`
- `allow-enumerate-tree`
- `allow-copy-document`
- `allow-check-storage-permission`
- `allow-open-app-settings`
